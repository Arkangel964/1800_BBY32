## General Info

This browser based web application is designed to allow gamers to connect, make new friends and have fun peri and post pandemic.

	
## Technologies
Technologies used for this project:
* HTML, CSS
* JavaScript
* Bootstrap 
* Firebase and Firestore
	
## Content
Content of the project folder:

```
 Top level of project folder: 
├── .gitignore               # Git ignore file
├── index.html               # landing HTML file, this is what users see when you come to url
├── chat.html                # chat HTML file, the page for chat
├── editprofile.html         # edit profile HTML, the user can change profile settings
├── friends.html             # friends HTML, user can see friends (not implemented)
├── games.html               # games HTML, user can choose games they play to display on their profile
├── login.html               # login HTML, log-in page
├── main.html                # main HTML, landing page after log-in and user setup
├── userprofile.html         # user profile HTML, personal profile for the currently signed in user
├── userreview.html          # user review HTML, review other users (not implemented)
└── README.md                # Read Me File
